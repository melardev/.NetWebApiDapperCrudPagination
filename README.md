# Introduction
This is a mini application implementing REST CRUD operations with Pagination using ASP.Net Web Api and Dapper( SQL and Stored procedures ).

# Getting Started
- This project uses MSSQL Server, run the setup.sql to create the database and table.
Then you would be ready to go with the API.
